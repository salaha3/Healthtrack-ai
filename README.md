HealthTrack AI

HealthTrack AI is a web-based fitness and nutrition tracking system developed as a final year project.

Features

- User profile with calorie and macro calculation
- Daily activity logging (steps, sleep, weight)
- Meal logging (calories, protein, carbs, fats)
- Dashboard with data visualisation
- Goal tracking (calories, protein, steps)

Technologies Used

- Node.js (Express)
- MySQL (Docker)
- EJS (Frontend templating)
- Chart.js (Data visualisation)
